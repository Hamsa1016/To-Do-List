# 📝 To-Do List Web App

A simple and interactive To-Do List application built using HTML, CSS, and JavaScript to manage daily tasks efficiently.

---

## 🚀 Features

- ➕ Add new tasks  
- ✅ Mark tasks as completed  
- ❌ Delete tasks  
- 💾 Data stored in browser (Local Storage)  
- 🎨 Clean and responsive UI  

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript  

---

## ▶️ How to Run

1. Download or clone the repository:
git clone https://github.com/your-username/to-do-app.git

2. Open the project folder

3. Double click `index.html`  
(or open with Live Server in VS Code)

---

## 💡 How It Works

- User enters a task and clicks **Add**
- Task is displayed in the list
- User can:
  - Mark as completed ✅
  - Delete ❌
- Tasks are saved using **Local Storage**, so they remain after refresh
